# FZ. – Personal Portfolio

A modern, responsive personal portfolio website for Fatima Zahra Lasfar, showcasing her skills, experience, and projects as a software engineer, data analyst, and designer.

## ✨ Features

- **Fully responsive** – adapts seamlessly to mobile, tablet, and desktop screens.
- **Dark / light mode toggle** – remembers user preference via `localStorage`.
- **Glassmorphism design** – subtle backdrop filters and soft shadows.
- **Profile picture** – circular image with hover effect.
- **Smooth scrolling** – anchor links scroll smoothly with offset for the sticky navbar.
- **Skills & technologies** – displayed as tags inside glass cards.
- **Language cards** – flags and proficiency levels.
- **Project showcase** – interactive cards with hover animations.
- **Contact form** – styled, non‑functional placeholder (can be connected to a backend).

## 🛠️ Technologies Used

- **HTML5** – semantic markup.
- **CSS3** – custom properties (variables), Flexbox, Grid, animations, media queries.
- **JavaScript (ES6)** – dark mode toggle with `localStorage`.
- **Font Awesome 6** – icons.
- **Google Fonts** – Inter typeface.
